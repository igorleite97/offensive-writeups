# Offensive Security Write-Ups

Documentação técnica de laboratórios de segurança ofensiva realizados
em ambientes controlados.

## Plataformas

- TryHackMe
- HackTheBox

## Objetivo

Este repositório documenta o processo de exploração de máquinas
vulneráveis, incluindo metodologia, ferramentas utilizadas e vetores
de ataque identificados.

## Skills Demonstradas

- Reconhecimento e enumeração
- Web exploitation
- Credential harvesting
- Privilege escalation
- Linux post-exploitation

## Rooms Documented
- Basic Pentesting
- Mr Robot CTF
- Wonderland
- Attacktive Directory
- Kenobi
- Basic Malware RE
- Lookup
